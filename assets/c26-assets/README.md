# C26 Website Assets

Source: `TUM-Blockchain-Club/c26-website`, commit `1ba8db1`.

This folder mirrors the C26 website `public/` asset tree, excluding the deployment-only `CNAME` file. The C26 app favicon from `app/favicon.ico` is stored at `app/favicon.ico`.

On this macOS filesystem, `public/student-initiatives/START-logo.png` and `public/student-initiatives/start-logo.png` cannot both exist in the same folder because their names differ only by case. The lower-case file is in `student-initiatives/start-logo.png`; the upper-case source file is stored separately at `case-collisions/student-initiatives/START-logo.png`.
